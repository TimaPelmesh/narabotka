# Альфа-Банк | Инвестиционный портал

Прототип веб-приложения для управления инвестиционным портфелем от Альфа-Банка.

## Описание

Инвестиционный дашборд, предоставляющий пользователям возможность отслеживать состояние своего портфеля, анализировать динамику активов и управлять инвестициями. Прототип демонстрирует основные функции и интерфейс будущего приложения.

## Технологический стек

- **HTML5** - структура и разметка
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript (Vanilla)** - интерактивность и динамика
- **Chart.js** - визуализация данных (графики и диаграммы)

## Основные функции

- Дашборд с обзором портфеля
- Статистика по активам и доходности
- Интерактивные графики динамики портфеля
- Распределение активов (акции, облигации, ETF)
- Список топ активов с детальной информацией
- Быстрые действия для управления инвестициями

## Дизайн

Приложение выполнено в корпоративной стилистике Альфа-Банка:
- **Цветовая палитра**: красный (#EF3124), белый, черный
- **Стиль**: минималистичный, современный
- **Типографика**: Inter (Google Fonts)
- **Адаптивность**: поддержка различных размеров экранов

## Запуск

Прототип представляет собой статический сайт и может быть открыт напрямую в браузере или размещен на GitHub Pages.

1. Клонировать репозиторий
2. Открыть `index.html` в браузере
3. Или разместить на GitHub Pages для онлайн-доступа
